# 快速上手指南

## 演示场景：两台手机互相发现

### 场景设置
- 手机 A: 命名为 "客厅手机"
- 手机 B: 命名为 "卧室手机"
- 两台手机连接到同一个 Wi-Fi

---

## 演示步骤

### 第一步：启动应用（两台手机）

```
┌─────────────────────────────┐
│      局域网设备发现          │
├─────────────────────────────┤
│                             │
│  [我的设备信息]              │

│  │ [客厅手机    ]      │    │
│  └─────────────────────┘    │
│                             │
│  [注册服务]  [注销服务]      │
│  状态: 未注册                │
│                             │
└─────────────────────────────┘
```

### 第二步：注册服务（两台手机都操作）

1. 手机 A 输入 "客厅手机"，点击【注册服务】
2. 手机 B 输入 "卧室手机"，点击【注册服务】
3. 状态显示变为：

```
状态: ✅ 已注册
```

### 第三步：开始发现（两台手机都操作）

点击【开始发现】按钮，状态变为：

```
发现的设备: 扫描中...
```

### 第四步：查看发现的设备

等待几秒钟后，设备列表会显示对方：

### 第五步：远程响铃测试

1. 在手机 A 上点击 "卧室手机" 的【响铃】按钮
2. 手机 B 立即开始播放铃声 💡
3. 手机 B 的界面显示：

4. 再次点击【响铃】按钮停止铃声

## 常见问题

### Q: 发现不了其他设备？
**A**: 检查以下几点：
1. 两台手机是否在同一 Wi-Fi？
2. 两台手机是否都点击了【注册服务】？
3. 是否有防火墙或路由器限制？

### Q: 点击响铃没反应？
**A**:
1. 检查目标设备是否已注册服务
2. 检查目标设备的媒体音量是否开启
3. 检查网络连接是否稳定

### Q: 如何同时让多台设备响铃？
**A**: 依次点击每个设备的【响铃】按钮即可。

## 技术演示要点

### mDNS 发现过程
```
手机 A                    局域网                    手机 B
  |                         |                         |
  |--- 发送服务注册 -------->|                         |
  |                         |<-------- 发送服务注册 ---|
  |                         |                         |
  |--- 发送发现请求 -------->|                         |
  |                         |-------- 发现响应 ------>|
  |<-------- 发现响应 ------|                         |
  |                         |                         |
  |                         |<-------- 发现响应 ------|
  |--- 解析服务信息 -------->|                         |
  |                         |-------- 服务详情 ------>|
```

### 响铃命令流程
```
手机 A (客户端)              手机 B (服务器)
     |                            |
     |--- TCP 连接 -------------->|
     |--- "RING\n" --------------->|
     |                            |---> 播放铃声
     |<--- 连接关闭 --------------|
     |                            |
     |--- TCP 连接 -------------->|
     |--- "STOP\n" -------------->|
     |                            |---> 停止铃声
     |<--- 连接关闭 --------------|
```
